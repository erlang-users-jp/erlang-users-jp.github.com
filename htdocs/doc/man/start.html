名前

start - UNIX用のOTP起動スクリプトのサンプルスクリプト  

説明

このドキュメントでは、Unix上で組み込みモードのErlangを起動するためのサンプルスクリプトであるstartについて説明します。

より詳細な使い方については、Embedded System User's Guideをご覧ください。

 

使い方

start [ data_file ]

data_file:
(オプション)start_erl.dataの代わりに使用するデータファイルを指定することができます。

環境変数RELDIRで、リリースファイル[?]のパスを指定することができます。

 

関連項目

run_erl(1), start_erl(1)
